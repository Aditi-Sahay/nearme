## Overview
A website for downloading the app: NearMe, designed to show you a list of all places around you using your IP address using a Google provided places API key based on the filters you apply.

## Tech Stack

#App:
- React Native
- Expo Go

#Website
- HTML
- Vercel

## Features
- Responsive design
- Interactive animations
- Project showcase
- App install link

## Challenges

A major challenge was to do it all for free, and since I was only 18 at the time, i had to optimize an create this website for downloading instead of launching it on Google Play Store as it required $25 for creating a developer's account.
